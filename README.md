# 项目简介
wepy开发的校园快递代拿小程序,ui框架使用了Vant weapp,功能包括微信授权登陆，编辑发布快递任务，接取快递任务，选择修改删除地址，分享任务，一共一个个页面，比较简单，适合小程序入门开发。

# 附录
后台接口用php写的 代码在根目录下php.zip里面，数据库文件也在里面
